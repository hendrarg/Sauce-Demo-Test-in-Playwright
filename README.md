# 🧪 Sauce-Demo-Test-in-Playwright

Automated testing project for [SauceDemo](https://www.saucedemo.com/v1/) using **Playwright with TypeScript**.

## 📋 Deskripsi

Project ini merupakan implementasi automation test berbasis Playwright untuk website [SauceDemo](https://www.saucedemo.com/v1/). Fokus pengujian adalah pada:

- ✅ **Conditional Login**  
- ✅ **Validasi jumlah produk dan detail produk**  
- ✅ **Simulasi penambahan produk ke cart dan validasi harga**

## 🔧 Teknologi

- [Playwright](https://playwright.dev/)
- TypeScript
- Node.js

---

## 📦 Instalasi

1. Clone repositori:
   ```bash
   git clone https://github.com/username/Sauce-Demo-Test-in-Playwright.git
   cd Sauce-Demo-Test-in-Playwright
2. Install dependencies:
   npm install
3. Jalankan test:
   npx playwright test
4. (Opsional) Jalankan di UI mode:
   npx playwright test --ui
